# Predictive Insight Engine – House Price Prediction

## Project Overview
The Predictive Insight Engine is a machine learning project developed to predict house prices using supervised learning algorithms. The project uses different regression techniques and evaluation metrics to analyze and improve prediction accuracy.

This project demonstrates:
- Simple Linear Regression
- Multiple Linear Regression
- Polynomial Regression
- Gradient Descent Optimization
- Bias–Variance Analysis

---

# Objective
The main objective of this project is to predict house prices based on various features such as:
- Area of house
- Number of bedrooms
- Number of bathrooms
- Location score
- Property age

The project also compares different regression models to identify the best-performing model.

# Machine Learning Algorithms Used

## 2. Multiple Linear Regression

## 3. Polynomial Regression

## 4. Gradient Descent

# Best Performing Model

Multiple Linear Regression performed best because:
- It used multiple important features.
- It reduced prediction error.
- It maintained a balance between bias and variance.
